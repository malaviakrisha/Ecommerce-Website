# Ecommerce Website

Welcome to my Ecommerce Website project! This is a fully responsive online shopping platform developed as part of my portfolio. It demonstrates key front-end skills in creating an engaging user interface for e-commerce applications.

## ✨ Features
- Responsive design that works on desktop, tablet, and mobile devices.
- Product catalog with categories, search, and filtering options.
- Shopping cart with add/remove items and total calculation.
- Product detail pages with images and descriptions.
- Clean navigation, footer, and contact sections.

## 🛠️ Tech Stack
- HTML5 for structure.
- CSS3 (with Flexbox/Grid or Bootstrap) for styling.
- JavaScript (vanilla or libraries like Glide.js) for interactivity.
- Possibly MongoDB Atlas for backend simulation (if applicable).
